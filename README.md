# asf-mainpage-2.0

### 运行前安装
```
yarn install
```

### 实时调试
```
yarn serve
```

### 打包
```
yarn build
```

### 代码检测
```
yarn lint
```

## Commit时备注
* 增加从未有过的功能，即新增功能性提交：`<feat>: +提交说明`
* 改变从前的功能，即不影响页面功能的提交：`<style>: +提交说明`
* 修复从前的功能，即新增修复性提交： `<fix>: +提交说明`
* 更新代码： `<update>: +提交说明`
* 提交说明可以使用中文

##变量命名规范
优先遵循eslint+babel自带规范，若无提示,默认使用骆驼命名法(camelCase)
